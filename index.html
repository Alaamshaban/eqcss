What is EQCSS

EQCSS is a JavaScript plugin that extends CSS to include Element Queries, plus other features that regular CSS may not support for years. It it written in pure JavaScript, meaning it doesn't require jQuery or any other JS library to be installed on your site to work.

How to use it

  • Install EQCSS at the end of your <body> with 
    <script src="EQCSS.js"></script>
    .
  • Write your EQCSS code in the HTML page with 
    <script type="text/eqcss"> ... </script>
    .
    Or include it from an external file with 
    <script type="text/eqcss" src="code.eqcss"></script>
    .
  • The 
    EQCSS.apply()
    function is called on page load and on resize. Call it manually when you update the DOM.
  • One EQCSS script can contain multiple queries, and a query can contain multiple conditions.
  • Queries have the following form: 
    @element {selector} and {condition} [ and {condition} ]* { {css} }
  • {css}
    can contain any valid CSS rules. Ex: "#id div { color: red }".
  • {selector}
    is a CSS selector targeting one or many elements. Ex: "#id" or ".class".
  • {condition}
    is composed of a measure and a value. Examples: 
    • min-width: 500px
      : condition is true if the element's width is 500px or more.
    • min-width: 50%
      : condition is true if the element's width is 50% or more of its parent's width.
    • max-width: 500px
      : condition is true if the element's width is 500px or less.
    • max-width: 50%
      : condition is true if the element's width is 50% or less of its parent's width.
    • min-characters: 50
      : condition is true if the element contains 50 or more characters.
    • max-characters: 50
      : condition is true if the element contains 50 or less characters.
    • min-children: 50
      : condition is true if the element has 50 or more direct children.
    • max-children: 50
      : condition is true if the element has 50 or less direct children.
